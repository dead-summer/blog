## 1.1 数组的概念

数组是一种容器，用于存储同一类型的多个数据。例如，可以使用 `int[]` 存储整数数据，使用 `String[]` 存储字符串数据。

示例：

* 存储整数：`int[] array = {20, 10, 80, 60, 90};`
* 存储字符串：`String[] names = {"牛二", "西门", "全蛋"};`

## 1.2 数组的应用场景

变量和数组都用于存储数据，但数组在处理批量数据时更具优势。
