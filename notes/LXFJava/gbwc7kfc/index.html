List жҳҜдёҖз§ҚеҹәзЎҖзҡ„йӣҶеҗҲзұ»еһӢпјҢе®ғд»ЈиЎЁдёҖдёӘжңүеәҸеҲ—иЎЁгҖӮList еңЁеҶ…йғЁжҢүз…§е…ғзҙ зҡ„ж·»еҠ йЎәеәҸеӯҳеӮЁпјҢе№¶дё”жҜҸдёӘе…ғзҙ йғҪеҸҜд»ҘйҖҡиҝҮзҙўеј•жқҘи®ҝй—®пјҢзҙўеј•д»Һ 0 ејҖе§ӢпјҢиҝҷдёҺж•°з»„зұ»дјјгҖӮ

2.1 List дёҺж•°з»„зҡ„еҜ№жҜ”

иҷҪз„¶ж•°з»„е’Ң List йғҪжҳҜжңүеәҸз»“жһ„пјҢдҪҶж•°з»„еңЁж·»еҠ е’ҢеҲ йҷӨе…ғзҙ ж—¶дёҚеӨҹзҒөжҙ»гҖӮдҫӢеҰӮпјҢд»Һж•°з»„дёӯеҲ йҷӨдёҖдёӘе…ғзҙ йңҖиҰҒз§»еҠЁеҗҺз»ӯе…ғзҙ пјҢиҖҢж·»еҠ е…ғзҙ еҲҷйңҖиҰҒе…Ҳз§»еҠЁзҺ°жңүе…ғзҙ д»Ҙи…ҫеҮәз©әй—ҙгҖӮиҝҷдәӣж“ҚдҪңеңЁж•°з»„дёӯе®һзҺ°иө·жқҘжҜ”иҫғйә»зғҰгҖӮ

еӣ жӯӨпјҢеңЁйңҖиҰҒйў‘з№ҒеўһеҲ е…ғзҙ зҡ„еңәжҷҜдёӢпјҢArrayList жҳҜжӣҙеёёз”Ёзҡ„йҖүжӢ©гҖӮArrayList еҶ…йғЁдҪҝз”Ёж•°з»„еӯҳеӮЁе…ғзҙ пјҢе№¶е°ҒиЈ…дәҶж·»еҠ е’ҢеҲ йҷӨзҡ„ж“ҚдҪңпјҢдҪҝеҫ—ж“ҚдҪң List зұ»дјјдәҺж“ҚдҪңж•°з»„пјҢиҖҢж— йңҖе…іеҝғеҶ…йғЁе…ғзҙ зҡ„з§»еҠЁгҖӮ

2.2 List жҺҘеҸЈзҡ„дё»иҰҒж–№жі•

List<E> жҺҘеҸЈжҸҗдҫӣдәҶдёҖзі»еҲ—ж–№жі•жқҘж“ҚдҪңеҲ—иЎЁдёӯзҡ„е…ғзҙ пјҡ

ж–№жі•жҸҸиҝ°
boolean add(E e)еңЁеҲ—иЎЁжң«е°ҫж·»еҠ дёҖдёӘе…ғзҙ гҖӮ
boolean add(int index, E e)еңЁжҢҮе®ҡзҙўеј•дҪҚзҪ®ж·»еҠ дёҖдёӘе…ғзҙ гҖӮ
E remove(int index)еҲ йҷӨжҢҮе®ҡзҙўеј•дҪҚзҪ®зҡ„е…ғзҙ гҖӮ
boolean remove(Object e)еҲ йҷӨжҢҮе®ҡзҡ„е…ғзҙ гҖӮ
E get(int index)иҺ·еҸ–жҢҮе®ҡзҙўеј•дҪҚзҪ®зҡ„е…ғзҙ гҖӮ
int size()иҺ·еҸ–еҲ—иЎЁзҡ„еӨ§е°ҸпјҲеҢ…еҗ«е…ғзҙ зҡ„дёӘж•°пјүгҖӮ

2.3 ArrayList е’Ң LinkedList зҡ„жҜ”иҫғ

ArrayList е’Ң LinkedList йғҪе®һзҺ°дәҶ List жҺҘеҸЈпјҢдҪҶе®ғд»¬зҡ„еҶ…йғЁе®һзҺ°ж–№ејҸдёҚеҗҢгҖӮArrayList еҹәдәҺж•°з»„е®һзҺ°пјҢиҖҢ LinkedList еҹәдәҺй“ҫиЎЁе®һзҺ°гҖӮ

иҺ·еҸ–жҢҮе®ҡе…ғзҙ йҖҹеәҰеҫҲеҝ«йңҖиҰҒд»ҺеӨҙејҖе§ӢжҹҘжүҫе…ғзҙ 
ж·»еҠ е…ғзҙ еҲ°жң«е°ҫйҖҹеәҰеҫҲеҝ«йҖҹеәҰеҫҲеҝ«
еңЁжҢҮе®ҡдҪҚзҪ®ж·»еҠ /еҲ йҷӨйңҖиҰҒз§»еҠЁе…ғзҙ дёҚйңҖиҰҒз§»еҠЁе…ғзҙ

2.4 List зҡ„зү№зӮ№

  • е…Ғи®ёйҮҚеӨҚе…ғзҙ : List е…Ғи®ёж·»еҠ йҮҚеӨҚзҡ„е…ғзҙ гҖӮ
  • е…Ғи®ё null еҖј: List е…Ғи®ёж·»еҠ  null еҖјгҖӮ

2.5 еҲӣе»ә List

йҷӨдәҶдҪҝз”Ё ArrayList е’Ң LinkedList д№ӢеӨ–пјҢиҝҳеҸҜд»ҘдҪҝз”Ё List жҺҘеҸЈжҸҗдҫӣзҡ„ of() ж–№жі•еҝ«йҖҹеҲӣе»ә Listпјҡ

List<Integer> list = List.of(1, 2, 5);

дҪҶжҳҜпјҢList.of() ж–№жі•дёҚжҺҘеҸ— null еҖјпјҢеҰӮжһңдј е…Ҙ nullпјҢдјҡжҠӣеҮә NullPointerException ејӮеёёгҖӮ

2.6 йҒҚеҺҶ List

йҒҚеҺҶ List жңүеӨҡз§Қж–№ејҸпјҡ

  1. дҪҝз”Ё for еҫӘзҺҜе’Ң get(int) ж–№жі•: иҷҪз„¶еҸҜд»ҘдҪҝз”Ё for еҫӘзҺҜе’Ң get(int) ж–№жі•жқҘйҒҚеҺҶ ListпјҢдҪҶиҝҷе№¶дёҚжҳҜжҺЁиҚҗзҡ„ж–№ејҸпјҢеӣ дёәе®ғеңЁ LinkedList дёӯж•ҲзҺҮиҫғдҪҺгҖӮ
  2. дҪҝз”Ёиҝӯд»ЈеҷЁ Iterator: йҖҡиҝҮиҝӯд»ЈеҷЁ Iterator йҒҚеҺҶ List жҳҜжңҖй«ҳж•Ҳзҡ„ж–№ејҸгҖӮ
    import java.util.Iterator;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<String> list = List.of("apple", "pear", "banana");
        for (Iterator<String> it = list.iterator(); it.hasNext(); ) {
            String s = it.next();
            System.out.println(s);
        }
    }
}
  3. дҪҝз”Ё for each еҫӘзҺҜ: for each еҫӘзҺҜжҳҜдҪҝз”Ёиҝӯд»ЈеҷЁ Iterator зҡ„иҜӯжі•зі–пјҢеҸҜд»Ҙз®ҖеҢ–д»Јз ҒгҖӮ
    import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<String> list = List.of("apple", "pear", "banana");
        for (String s : list) {
            System.out.println(s);
        }
    }
}

2.7 List е’Ң Array иҪ¬жҚў

  1. List иҪ¬жҚўдёә Array:
    • toArray(): иҝ”еӣһ Object[] ж•°з»„пјҢдјҡдёўеӨұзұ»еһӢдҝЎжҒҜпјҢе®һйҷ…еә”з”Ёиҫғе°‘гҖӮ
    • toArray(T[]): дј е…ҘдёҖдёӘзұ»еһӢзӣёеҗҢзҡ„ж•°з»„пјҢList еҶ…йғЁиҮӘеҠЁжҠҠе…ғзҙ еӨҚеҲ¶еҲ°дј е…Ҙзҡ„ж•°з»„дёӯгҖӮеҰӮжһңдј е…Ҙзҡ„ж•°з»„дёҚеӨҹеӨ§пјҢйӮЈд№Ҳ List еҶ…йғЁдјҡеҲӣе»әдёҖдёӘж–°зҡ„еҲҡеҘҪеӨҹеӨ§зҡ„ж•°з»„пјҢеЎ«е……еҗҺиҝ”еӣһпјӣеҰӮжһңдј е…Ҙзҡ„ж•°з»„жҜ” List е…ғзҙ иҝҳиҰҒеӨҡпјҢйӮЈд№ҲеЎ«е……е®Ңе…ғзҙ еҗҺпјҢеү©дёӢзҡ„ж•°з»„е…ғзҙ дёҖеҫӢеЎ«е…… nullгҖӮжңҖеёёз”Ёзҡ„ж–№ејҸжҳҜдј е…ҘдёҖдёӘвҖңжҒ°еҘҪвҖқеӨ§е°Ҹзҡ„ж•°з»„пјҡ
      Integer[] array = list.toArray(new Integer[list.size()]);
    • toArray(IntFunction<T[]> generator): жӣҙз®ҖжҙҒзҡ„еҶҷжі•пјҢдҪҝз”ЁеҮҪж•°ејҸжҺҘеҸЈгҖӮ
      Integer[] array = list.toArray(Integer[]::new);
  2. Array иҪ¬жҚўдёә List:
    • List.of(T...): е°Ҷж•°з»„иҪ¬жҚўдёә ListгҖӮ
    • Arrays.asList(T...): еңЁ JDK 11 д№ӢеүҚзҡ„зүҲжң¬дёӯдҪҝз”ЁгҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢдҪҝз”Ё List.of() иҝ”еӣһзҡ„жҳҜдёҖдёӘеҸӘиҜ» ListпјҢдёҚиғҪиҝӣиЎҢж·»еҠ жҲ–еҲ йҷӨж“ҚдҪңгҖӮ
